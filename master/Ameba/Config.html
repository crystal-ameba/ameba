    class Ameba::Config

    • Ameba::Config
    • Reference
    • Object

    Overview

    A configuration entry for Ameba::Runner.

    Config can be loaded from configuration YAML file and adjusted.

    config = Ameba::Config.load
config.formatter = my_formatter
